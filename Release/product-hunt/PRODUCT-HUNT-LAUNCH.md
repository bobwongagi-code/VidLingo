## Maker First Comment

Hey Product Hunt,

I built AirTranslate because I kept running into the same awkward problem: the audio I needed to understand was already playing on my Mac, but most transcription workflows still expected microphone routing, virtual cables, or a meeting-specific setup.

AirTranslate captures Mac system audio directly, turns it into a live transcript, and translates it in real time. You can keep the source and translation side by side, or use the floating caption window over a lecture, video, interview, stream, or meeting.

The default path uses Apple Speech and Apple Translation frameworks. GPT realtime transcription and translation are optional for people who want them, and AirTranslate only enables those modes after you add your own OpenAI API key. Keys stay in macOS Keychain, and saved transcripts are plain text files on your Mac.

This is an open-source macOS app, still early but already useful for multilingual listening workflows. I would love feedback on the capture flow, caption readability, and which language pairs or accessibility features would make it more useful.
